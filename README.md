# Twitter Clone
Twitter Clone created with React & Firebase.

[Live Demo](https://itsaflamingo.github.io/twitter-clone/) :point_left:

### Features

- Google sign in
- Create and delete tweets & retweets
- Individual user profiles
- Follow/Unfollow
- Sign out/Delete account 
- Search tweets
- Responsive design
